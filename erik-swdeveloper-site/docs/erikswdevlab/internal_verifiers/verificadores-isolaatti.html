Hola, futuro verificador interno

Si estás aquí es porque te interesa ser verificador de Isolaatti para Android. Sigue los siguientes pasos.

  1. Mándame tu correo de Google (@gmail.com) a mi correo erik10cavazos@gmail.com o dímelo de alguna otra forma.
  2. Espera a que te responda que fuiste agregado. Te proporcionaré un grupo de Facebook, ahí informarás los errores ;)
  3. Debes aceptar la invitación para pruebas internas. Invitación.
  4. Es posible que debas esperar hasta 48 horas para poder descargar la aplicación.

Retos de desarrollar para Android

Isolaatti para Android se encuentra en constante desarrollo, y por eso no está exento de errores. Aunado a ello, la plataforma Android impone muchos retos para los desarrolladores, pues no se tiene que soportar solo las ultimas 2 o 3 versiones de Android, sino 7 u 8 (dependiendo de que tanto se quiera abarcar), además de eso, existe una infinidad de dispositivos diferentes, con configuraciones de hardware y software distintas.

¡Tu ayuda es muy importante!

¡Entre más personas prueben esta aplicación, más errores puedo encontrar y corregir!

Cuando ejecutes la aplicación en tu dispositivo y la aplicación tenga un fallo catastrófico, esta me informará el error, brindándome datos relevantes para solucionar el error. Esto es transparente para ti, solo debes usar la aplicación como si de cualquier otra aplicación se tratara.

Aquí está el truco. Tu dispositivo tendrá una configuración diferente al mío o al emulador que pueda utilizar, por lo que el comportamiento puede variar.

También, puedes sugerir cambios o nuevas características.
